ИНСТРУКЦИЯ (Windows + Vite)

1) Скопируйте App.tsx в ваш проект, заменив существующий:
   .\src\App.tsx

2) Подготовьте локальные фото:
   - Создайте папку: .\public\gallery
   - Положите туда файлы 01.jpg ... 16.jpg (можно меньше — тогда удалите лишние строки в массиве galleryImages).
   - Пути в коде уже прописаны как /gallery/NN.jpg

   Пример PowerShell:
     mkdir .\public\gallery
     Copy-Item "C:\Путь\к\вашим\фото\*.jpg" ".\public\gallery\"

3) Если импорт из src/assets не нужен — файл images.d.ts можно не использовать.
   Я приложил "чистую" версию images.d.ts на случай, если захотите импортировать изображения как модули.

4) Запуск dev-сервера:
     npm run dev
   Откройте http://localhost:5173

5) Если не видно фото:
   - Проверьте, что файлы реально лежат в .\public\gallery
   - Имена и расширения (например 01.jpg), регистр символов важен.
   - После добавления новых файлов иногда помогает перезапуск dev-сервера.
